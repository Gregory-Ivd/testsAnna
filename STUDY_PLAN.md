# Воронка підготовки до НМТ — англійська + математика

Щоденний дріп тренувальних тестів у форматі реального НМТ-2026.
Мета: за один прохід воронки закрити **всі теми** НМТ з англійської та математики.

## Як це працює (для випускника)

1. Заходить на лендинг `index.html` → лишає контакт у формі → відкриваються тести.
2. **Щодня публікуємо 3 нові тести:** 2 з англійської + 1 з математики.
3. Кожен тест — у форматі НМТ, з таймером, ключами, міні-лекціями (💡) і детальним розбором після здачі.
4. Кожен результат летить у Google Sheets (`action=result`) + опитування «що далі» (`action=survey`).

## Конвенції для нових тестів (клонувати, не винаходити)

- **Англійський тренажер** → клон `nmt_english_vocab_day4.html` (MC-скелет):
  `ol.qs > li` з `data-correct` / `data-topic` / `data-sol`, об'єкти `hints` (міні-лекції по темах) і `sols` (розбір кожного питання), `TOTAL`, мапа `blocks`, вердикт.
- **Математичний тренажер** → клон `nmt_math_real.html`: Частина 1 — MC (`data-correct`=літера), Частина 2 — коротка відповідь (`input type=text`, `data-correct`=число, звірка через `numEqual`/`normalizeNum`).
- **Файл:** `nmt_<subject>_<slug>.html` у корені. **testId:** `english-day5-tenses`, `math-day5-numbers` і т.д.
- **Текст гендерно-нейтральний**, без зашитого імені — звертання лише через `taHey()/taVoc()` (кличний відмінок з форми). Уникати дієслів минулого часу 2-ї особи (вони мають рід).
- **Трекінг** (`TA.endpoint`, `taTrackResult`, `taRenderSurvey`) копіюється без змін — лише міняється `testId`.
- Після додавання тесту — **картка в `index.html`** (+ оновити лічильник «N тестів») і чекбокс у цьому файлі.
